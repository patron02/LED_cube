# LED_cube
Materials:
- 64 white LEDs
- Arduino UNO
- Arduino UNO shield
- 4 2N222 transistors 
- 4 1.2 [kOhm] resistors 
- 16 220 [Ohm] resistors 
- jumper cables 
- 32 AWG wire 
- 3D printed base
